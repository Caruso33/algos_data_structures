## Big O

### Comparison built-in Array and Objects

| Structure | Insertion   | Removal     | Searching | Access | Comment                                   |
| --------- | ----------- | ----------- | --------- | ------ | ----------------------------------------- |
| Array     | O(1) / O(n) | O(1) / O(n) | O(n)      | O(1)   | Ordered, depends operated at start or end |
| Object    | O(1)        | O(1)        | O(n)      | O(1)   | Unordered                                 |
